# Support

**No Sleep Pro** - Keep Your Mac Awake

## Getting Help

### Common Issues

**App not preventing sleep?**
- Make sure the app is active (filled icon in menu bar)
- Check if Battery Protection stopped it (Settings > Automation)
- Verify no conflicting sleep settings in System Settings > Energy

**Mouse Jiggler not working?**
- Grant Accessibility permission: System Settings > Privacy & Security > Accessibility
- Enable No Sleep Pro in the list

**Keyboard shortcut not responding?**
- Grant Accessibility permission (same as above)
- Check for conflicts with other apps using the same shortcut

**Menu bar icon missing?**
- The app may have quit - relaunch from Applications
- Check if "Launch at Login" is enabled in Settings

### Report a Bug

Found a bug? Please open an issue on GitHub with:
- macOS version
- App version (shown in About window)
- Steps to reproduce the issue
- Expected vs actual behavior

### Feature Requests

Have an idea? Open a feature request on GitHub and describe what you'd like to see.
