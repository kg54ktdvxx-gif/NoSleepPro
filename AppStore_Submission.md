AwakeApp is the most thoughtful menu bar utility for preventing your Mac from sleeping. Whether you're in a meeting, giving a presentation, or running long tasks, AwakeApp keeps your Mac awake intelligently.

- Schedule-based activation - Stay awake during work hours automatically
- Wi-Fi network triggers - Activate when connected to office or home network
- Hardware triggers - Activate when power adapter or external display connected
- Battery protection - Automatically stops when battery drops below your threshold
- Global keyboard shortcut (Cmd+Shift+A) - Toggle from anywhere, even in fullscreen apps

NEW IN VERSION 1.2: CLOSED-LID MODE
- Keep your Mac awake with the lid closed (clamshell mode)

FLEXIBLE TIMER OPTIONS

- 1 hour - Presentations

- 5 hours - Long downloads or renders

POWER USER FEATURES
- Mouse jiggler keeps you "Active" in chat apps like Slack and Teams

PERFECT FOR
- Video calls and presentations
- Large file downloads

- Remote desktop sessions
- Developers running long builds
- Desktop setups with closed MacBook
- Anyone who's been interrupted by unexpected sleep

BATTERY FRIENDLY

PRIVACY FOCUSED

## App Store Review Notes

```
AwakeApp prevents Mac from sleeping using IOPMAssertionCreateWithName (IOKit).

KEY FEATURES TO TEST:

1. Basic Timer:
   - Click menu bar icon -> Select duration -> Timer starts
   - Countdown visible in menu bar and dropdown
   - Mac won't sleep while active

2. Keyboard Shortcut (Cmd+Shift+A):
   - Press Cmd+Shift+A anywhere to toggle on/off
   - Works even in fullscreen apps

3. App Triggers:
   - Open Settings -> App Triggers tab
   - Enable Zoom trigger
   - Launch Zoom -> AwakeApp auto-activates
   - Quit Zoom -> AwakeApp auto-deactivates

4. Schedule:
   - Open Settings -> Schedule tab
   - Add a schedule for current time
   - AwakeApp activates automatically

5. Battery Protection:
   - Open Settings -> Automation tab
   - Enable battery protection (set threshold)
   - When on battery below threshold, timer stops
   - Shows notification message

6. Closed-Lid Mode (NEW in v1.2):
   - Open Settings -> Automation tab -> Closed-Lid Mode section
   - Status shows if requirements are met (power + display)
   - Enable when requirements met
   - Close MacBook lid -> Mac stays awake

7. Mouse Jiggler:
   - Requires Accessibility permission
   - Open Settings -> Automation tab
   - Enable mouse jiggler
   - Cursor moves slightly every 60 seconds

The app uses no special entitlements beyond App Sandbox.
No account or login required.
No network access required (except for Wi-Fi SSID detection).
```
